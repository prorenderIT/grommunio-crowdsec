## 🔐 grommunio-crowdsec

**Automated CrowdSec integration for grommunio**

Protect your grommunio mail server against **brute-force attacks and login abuse** with minimal setup.

---

## ⚠️ Why this matters

Unprotected mail servers are constantly targeted by automated attacks.
This can lead to:

* Compromised mail accounts
* Spam being sent from your server
* Blacklisting of your domain
* Increased administrative overhead

---

## 🚀 What this project does

This script installs and configures CrowdSec to work seamlessly with grommunio services.

### 🔧 Features

* Automated CrowdSec installation
* Integration with:

  * Postfix
  * IMAP
  * ActiveSync
  * DAV services
* Detection of authentication failures
* Brute-force protection
* Simple setup for openSUSE

---

## ⚡ Quick Start

```bash
chmod +x ./install_CrowdSec_for_grommunio.sh
./install_CrowdSec_for_grommunio.sh
```

* grommunio environment
